

## Cyber Defence

First, clone this repository and run the development server:

```bash
$ git clone git@github.com:ishan-im/Cyber-defence.git
npm install
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
